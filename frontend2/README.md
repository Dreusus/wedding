# Свадебное приглашение - Павел и Валерия

Этот проект содержит оригинальный HTML сайт-приглашение, скачанный с https://inviteforwedd.ru/pavelandvaleria/package2

## Структура проекта

```
frontend2/
├── public/
│   ├── wedding.html           # Оригинальный HTML (1:1 копия package2.html)
│   ├── static.tildacdn.com/   # CSS, JS, изображения с Tilda CDN
│   ├── neo.tildacdn.com/      # Скрипты Tilda
│   ├── megatimer.ru/          # Скрипты таймера
│   └── images/                # Дополнительные изображения
├── src/
│   ├── App.tsx                # Простой редирект на wedding.html
│   └── main.tsx               # Точка входа
└── index.html                 # HTML шаблон
```

## Как это работает

- При запуске приложения `App.tsx` делает редирект на `/wedding.html`
- Файл `wedding.html` - это точная копия скачанного сайта с package2
- Все пути к ресурсам (CSS, JS, изображения) исправлены для работы локально
- Никакие React компоненты не используются - сайт отображается как оригинальный HTML

## Особенности

- ✅ **Оригинальный HTML без изменений** - сайт Tilda используется как есть
- ✅ **Все ресурсы локально** - CSS, JS и изображения скопированы из скачанного сайта
- ✅ **Точная копия дизайна** - полное соответствие оригиналу
- ✅ **Цвета**:
  - Бордовый: `#4b0b22`
  - Кремовый: `#f6ece3`
  - Желтый: `#fff705`
- ✅ **Шрифт Montserrat** с поддержкой кириллицы
- ✅ **Tilda анимации** - все оригинальные анимации работают
- ✅ **Адаптивная верстка** - дизайн адаптируется под все устройства

## Запуск

```bash
# Установка зависимостей
npm install

# Запуск dev сервера
npm run dev

# Сборка для production
npm run build
```

Сайт будет доступен на: **http://localhost:5175/**

## Технологии

- React 19 (только для базовой структуры проекта)
- TypeScript
- Vite
- Оригинальный HTML/CSS/JS с Tilda

## Примечание

Этот проект не использует React компоненты для рендеринга контента. Весь контент берётся из оригинального HTML файла, скачанного с сайта inviteforwedd.ru. React используется только как обёртка для удобства разработки и деплоя.
